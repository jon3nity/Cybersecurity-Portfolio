Floor Convention (Irish)

  • L21st Floor — MDF ★ Rm 2284 + 3 IDFs
  • L32nd Floor — 3 IDFs (3728, 3337, 3352)

  • Fibre riser: exactly 1 floor each direction

Packet Tracer Devices

  • MDF1× Cisco 3560 L3 → MDF-SW Rm 2284
  • L13× Cisco 2960 → L1-IDF1 / IDF2 / IDF3
  • L23× Cisco 2960 → L2-IDF1 / IDF2 / IDF3
  • L33× Cisco 2960 → L3-IDF1 / IDF2 / IDF3
  • CL3× Cisco 2960 → CoLab switches
  • 1× Router → Edge Router (WAN)

Cabling Rules

  • MDF ↔ same-floor IDFs: copper crossover, Gi ports
  • MDF ↔ other floors: multimode fibre riser
  • Interbuilding MDF ↔ CoLab: fibre
  • IDF ↔ PC: straight-through Cat6
  • 90mMax horizontal run per IEEE std
  • All inter-switch links: 802.1Q trunk, native VLAN 99
